# alarm

macOS 原生周期提醒应用。运行时在桌面置顶显示圆形进度悬浮窗（随时间逐渐填满），并按设定间隔触发提醒（弹窗、语音、屏幕柔光闪烁可单独开关）。

## 功能

- 常驻圆形悬浮窗：圆环随周期推进逐渐填满，中央显示倒计时，可拖动；到点时加强动效
- 悬浮窗配色主题：41 款预设，进度环 / 倒计时 / 菜单按钮可独立配色（含霓虹、终端、朝霞、极光等三色主题），适配纯黑与纯白壁纸；支持自定义各部位颜色
- 可配置提醒间隔（默认 10 分钟；预设 5/10/15/25/45/60 分钟或自定义 1–180 分钟）
- 提醒方式：弹窗、TTS 语音、屏幕柔光闪烁（白色圆角光晕呼吸，可多选）
- 菜单栏快捷操作：暂停/继续、打开设置
- 关闭设置窗口后应用继续运行；完全退出（Cmd+Q）后停止提醒

## 构建与运行

1. 使用 Xcode 26+ 打开 `alarm.xcodeproj`
2. 选择 **alarm** scheme，运行（⌘R）

或使用命令行：

```bash
xcodebuild -scheme alarm -configuration Debug build
open build/Debug/alarm.app
```

## 性能与内存

- 调度：单次 `fireTimer` 到点触发；系统唤醒后自动重排
- 悬浮窗：倒计时每秒更新；圆环进度在剩余大于 2 分钟时每 10 秒、大于 1 分钟每 5 秒刷新
- 柔光闪烁：AppKit 圆角描边 + 白色光晕，按秒预调度约 9 次更新，中心不遮挡
- 设置窗：打开/激活时刷新剩余时间，无后台 TimelineView；可切换悬浮窗主题并预览
- 语音：懒加载 `AVSpeechSynthesizer`，播报结束释放
- 可用「活动监视器」对比空闲、设置窗打开、柔光闪烁前后的 CPU/内存

## 说明

- 提醒仅在应用进程运行时有效（无系统通知、无登录项）
- 部署目标：macOS 26.2（可在 Xcode 中下调以兼容更早系统）
- 应用已启用 App Sandbox，语音使用系统 `AVSpeechSynthesizer`，无需额外权限
